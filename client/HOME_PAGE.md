# LowXena Game - Home Page Features

## New Home Page

The application now features a stunning home page with the following elements:

### Features:

1. **Animated Background**
   - Floating stars animation
   - Rising particles effect
   - Pulsing gradient overlay
   - Smooth transitions and animations

2. **Play Button (Bottom Left)**
   - Large, animated play button with gradient background
   - Pulsing play icon
   - Glowing hover effects
   - Navigates to the game page when clicked

3. **Profile Section (Top Right)**
   - User avatar display
   - Player name and level
   - Glass-morphism effect with backdrop blur
   - Hover animations and glow effects

4. **Game Title (Center)**
   - Large animated title "LowXena"
   - Gradient text with glow effect
   - Subtitle "Enter the Arena"

### Routes:
- `/` - Home page (landing page)
- `/game` - Game page (loads when play button is clicked)

### How to Run:
```bash
cd client
npm install
npm run dev
```

The app will open at `http://localhost:5173` (or the next available port).

### Technologies Used:
- React
- React Router DOM
- CSS3 Animations
- Gradient effects
- Backdrop filters
